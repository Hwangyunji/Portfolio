## 기능 소개
- 풀페이지 구성의 화면 전환
- 메인 페이지의 CSS와 자바스크립트를 이용한 타이핑 효과
- 좌우로 클릭하면 볼 수 있는 작업물들
- 페이지 바로가기, 상세정보 클릭시 새창으로 열림
- 전화번호와 메일 클릭시 통화 연결과 메일 보내기 가능
